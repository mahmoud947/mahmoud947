## 🚀 Featured Project

### [KPDF](https://github.com/mahmoud947/KPDF)

A Kotlin Multiplatform PDF library for Android and iOS with a Compose Multiplatform viewer layer.

**Highlights:**

- URL, Base64, bytes, and resource-backed PDF sources
- RAM page cache and disk page cache
- Remote source persistence for offline reopen
- Configurable page preloading
- Shared zoom state
- Save/export flow
- Open-in-external-app flow
- Open-from-device flow
- Connected toolbar and thumbnail strip
- Android and iOS Compose support

📚 Documentation: [KPDF Docs](https://mahmoud-b28887f9.mintlify.app/)
